Découverte du framework ANGULAR grâce à la plateforme de Dyma.FR

Première étape - Installer la CLI d'Angular

Une fois NodeJS installé, nous pouvons utiliser la commande npm afin de créer notre application angular.
Pour celà, nous allons utiliser dans notre terminal, cette ligne de commande :

npm install -g @angular/cli
Important : -g permet une installation globale d'Angular

Deuxième étape - Installer une app Angular

Une fois le CLI d'ANGULAR installé, nous pouvons créer notre application angular part le biais d'une nouvelle commande à exécuter dans notre terminal :

ng new myApp
Important : myApp correspond au nom de mon application
Deux étapes supplémentaire vous serons demandé :
  1. La première, c'est de savoir si l'on souhaite utilisé le routing
  2. La seconde c'est de savoir si l'on souhaite utiliser le CSS / SCSS ou rien du tout

Troisième étape - Se rendre dans notre app...

Une fois le CLI d'ANGULAR installé, nous devons créer notre application angular via le biais d'une commande à exécuter dans notre terminal.

cd myApp
Important : cd veut simplement dire qu'on change de répertoire.Ici nous aurons myApp/ comme dossier principal de notre application.

Quatrième étape - Simuler un serveur locale...

Une fois nos 3 étapes terminer, il nous reste plus qu'à voir le contenu des fichiers en lançant un serveur locale qui compilera et interprêtera intégralement notre application.

ng serve
Important : Obligatoirement, le serveur sera démarrer sur : http://localhost:4200

Si l'on souhaite utiliser un autre serveur locale, nous ajouterons le paramètre --port suivie d'un numéro de port accessible.
Exemple :

ng serve --port 5555
Ainsi, le serveur locale sera accessible à cette adresse : http://localhost:5555
