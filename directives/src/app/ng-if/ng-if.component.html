*ngIf

Les affichages possible :
  • *ngIf="unContenu" renverra true
  • *ngIf="1" renverra true

  • *ngIf="" renverra false
  • *ngIf="0" renverra false
  • *ngIf="null" renverra false
  • *ngIf="undefined" renverra false
Description

Je créer un bouton qui au clic inversera sa valeure booléenne afin d'afficher un des paragraphes ci-dessous.

Paragraphe 1 : Visible si la valeur est à true

{{ paragraphe1 }}

Paragraphe 2 : Visible si la valeur est à false

{{ paragraphe2}}
