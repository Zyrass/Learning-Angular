Affiché le contenu situé entre un sélecteur.

Voici le contenu de mon app-enfant. J'affiche le contenu situé entre mon selecteur app-enfant où je veux dans ma page. A savoir que je peux seulement l'affiché qu'une seule fois.

    Le placement du contenu sera lui très important. Donc selon les informations que l'on saisit, il nous est possible de placer notre contenu où l'on veut.
    A savoir tout de même qu'un seul et même ng-content peut-être affiché. Ainsi, si nous saisissons 2x le même ng-content, le dernier ng-content sera lui pris en compte.

    Je suis un 1er texte qui n'a aussi aucun sens. Je proviens d'un tag : div Il est lui visible grâce à ng-content select="div" située dans le fichier enfant.component.ts du répertoire enfant.

    Je suis un 2ème texte qui n'a lui aussi aucun sens. Je proviens cet fois d'un attribut : jeSuisUnAttribut Il est lui visible grâce à ng-content select="jeSuisUneAttribut" située dans le fichier enfant.component.ts du répertoire enfant.

    Je suis un 3ème texte qui n'a aucun sens. Mais je proviens d'une classe Il est visible grâce à ng-content select=".jeSuisUneClasse" situé dans mon le fichier enfant.component.ts du répertoire enfant.
