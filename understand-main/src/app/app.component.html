index.html

Dans mon fichier index.html, j'ajouterai uniquement :

# Nom description
1 CSS de Bootstrap Framework CSS très connu. (Facilite le développement des applications)
2 Scripts JS de Bootstrap Permet d'interrargir avec les différents composants dynamiquement.
3 Font Google Selon le besoin nous pourrons ajouter ici des fonts Google afin de changer le style d'écriture
4 Font-Awesome Permet d'ajouter un très grand nombre d'icônes très facilement.

app.component.ts

Ici, nous définissons le selecteur qui pour mon application, elle correspond à app-root. Nous définissons ici le HTML directement ou via un fichier externe (dans notre cas dans un fichier externe). Mais il est aussi possible de mettre du code CSS ici. (Également pour nous c'est dans un autre fichier). Et pour finir jous exportons notre class qui est utilisé dans notre fichier app.module.ts

app.component.html

Ici nous auront le code HTML ainsi que les interpolations et le reste du contenu spécifique lié à notre application. A savoir que ce fichier représente le code que sera interprêté et qui sera affiché à l'écran.

app.component.css

C'est très simple, ici et uniquement ici, le code sera interprêté uniquement pour le composant app
